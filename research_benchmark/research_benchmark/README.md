# Checked-In Result Artifacts

This directory contains the paper-facing result artifacts that are intentionally versioned with the repository.

## Contents

- `results_hardware/`: hardware and backend result folders, summaries, traces, plots, and derived tables.
- `results_simulator/`: simulator result folders and summaries.
- `simulator_checkpoints/`: resume metadata for simulator sweeps.

These files are not temporary scratch output. They are preserved so readers can inspect the details behind the paper companion repository without re-running every experiment first.

For the broader layout, see [../../docs/PROJECT_LAYOUT.md](../../docs/PROJECT_LAYOUT.md).
