# Raw Log Archive

This directory preserves raw run logs from exploratory and longer benchmark sweeps.

## Layout

- `vqe_logs/`
- `cvar_vqe_logs/`
- `qaoa_logs/`
- `ws_qaoa_logs/`
- `ma_qaoa_logs/`
- `pce_logs/`
- `qrao_logs/`

These logs are kept for traceability and historical inspection. They are separate from the structured `result.json`, `summary.json`, and `trace.jsonl` artifacts written by the packaged workflow.
